# Field Management System — Team API

This document describes the REST API endpoints available to competing teams. All endpoints are read-only (`GET`) and return JSON.

**Base URL:** `http://10.10.1.1:8713/api/v1`

> **Note:** No authentication is required.

---

## Endpoints

### `GET /match`

Returns information about the current upcoming or live match.

**Responses**

| Status | Description |
|--------|-------------|
| `200 OK` | A match is live or upcoming. Returns match parameters. |
| `400 Bad Request` | No live or upcoming match. Returns an empty object `{}`. |

**200 Response Body**

```json
{
    "requiredVoltage": 6,
    "requiredRpm": 30
}
```

| Field | Type | Description |
|-------|------|-------------|
| `requiredVoltage` | `number` | The voltage (in volts) your robot must apply during the match. |
| `requiredRpm` | `number` | The RPM your robot must generate during the match. |

---

### `GET /time`

Returns the time remaining in the current upcoming or live match.

> **Note:** If a match is upcoming (not yet started), `time_s` and `time_ms` will always reflect the full match duration of **4 minutes**.

**Responses**

| Status | Description |
|--------|-------------|
| `200 OK` | A match is live or upcoming. Returns time remaining. |
| `400 Bad Request` | No live or upcoming match. Returns an empty object `{}`. |

**200 Response Body**

```json
{
    "time_s": 240,
    "time_ms": 240000
}
```

| Field | Type | Description |
|-------|------|-------------|
| `time_s` | `number` | Time remaining in the match, in **seconds**. `240` if the match has not started. |
| `time_ms` | `number` | Time remaining in the match, in **milliseconds**. `240000` if the match has not started. |

---

## Error Handling

When there is no live or upcoming match, all endpoints return HTTP `400` with an empty JSON object:

```json
{}
```

Your code should always check the HTTP status code before attempting to parse the response body.

---

## Example Usage

**Check if a match is active, then read its parameters:**

```python
import requests

BASE_URL = "http://10.10.1.1:8713/api/v1"

match = requests.get(f"{BASE_URL}/match")
if match.status_code == 200:
    data = match.json()
    print(f"Required Voltage: {data['requiredVoltage']}V")
    print(f"Required RPM: {data['requiredRpm']}")
else:
    print("No active match.")
```
